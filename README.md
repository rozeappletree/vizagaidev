### Linking domain to gh pages

add `A record` with `@` host in DNS records.

```
A  @  185.199.108.153  AUTO TTL
A  @  185.199.109.153  AUTO TTL
A  @  185.199.110.153  AUTO TTL
A  @  185.199.111.153  AUTO TTL
```
